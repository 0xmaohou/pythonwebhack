# pythonwebhack

> 用python 2.7实现的web框架建立的在线渗透平台
> web框架是flask  前端框架是amazeUI

## 安装
`pip install flask`
`pip install requests`
`pip install MySQLdb  `

[usage]: python `myweb.py `

乌云数据库文件安装 链接: http://pan.baidu.com/s/1hrKYy8W 密码: yrrr
